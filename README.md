The Expense Tracker & Analytics project is a Python-based application designed to help users manage, analyze, and visualize their income and expenses efficiently. The system allows users to record financial transactions, categorize them, generate monthly reports using SQL aggregation, detect unusual spending patterns, and visualize expenses through charts.

This project focuses on data handling, analytics, and business insights, making it suitable for backend and analytics-oriented roles.
